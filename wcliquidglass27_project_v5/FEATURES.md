# WCLiquidGlass27 v5 功能说明

这是按截图风格重新设计的设置页版本。工程为独立重写，不包含原插件授权、联网校验、设备绑定逻辑。

## 设置页呈现

- 深色蓝紫渐变背景
- 顶部大标题 `LiquidGlass 设置`
- 顶部玻璃信息卡片
- 蓝紫色功能标签：Liquid Glass / iOS26+ / iOS27 Ready
- 分组标题前带彩色圆点
- 每行采用半透明玻璃卡片背景
- 开关使用系统 `UISwitch`
- 强度使用 `UISlider`
- 颜色使用 iOS 14+ 系统颜色选择器，低版本走 RGB 滑条回退页

## 配置清理

- `恢复默认设置`：只恢复本插件当前版本的默认开关、强度和颜色。
- `清除插件配置并关闭`：扫描并删除 `wclg27_`、`WCLG`、`WeChatLiquidGlass` 等插件相关配置键，然后把总开关设为关闭。
- 外部清理标记：如果设置页进不去，可以创建 `/var/mobile/Library/Preferences/wclg27_clear_config`，下次微信启动时自动清理配置并关闭插件。

## 入口

- 微信 → 我 → 插件管理 → WeChatLiquidGlass
- 长按底部 TabBar 右侧搜索按钮打开设置页
- 长按首页右上角搜索按钮打开设置页
